核心特性

让 Git 如此简单

不用再背命令 —— 所有操作都通过直观的图形界面完成。

📂

工作区目录树

以树形结构展示所有文件及其 Git 状态，不同颜色标识已修改、未跟踪、已暂存、忽略等状态。展开或收起子目录，一目了然。

📥

双击暂存

在工和区双击文件即可暂存，双击暂存区的文件又可取消暂存。一键「暂存所有」或「取消暂存」，流畅高效。

💾

一键提交

填写提交说明，点击提交即可创建快照。提交历史清䘏展示，支持按版本一键恢复工作区文件。

🆕

新建 Git 仓库

在非 Git 仓库目录下自动提��一键初始化。再也不需要在命令行敲 git init。

🌐

远程管理（Gitee + GitHub）

统一管理 Gitee 和 GitHub 远程仓库。分别配置 SSH 地址、一键检测连接、各自拉取推送，还支持一条命令同时推送到两个平台。

🔍

Git 状态全量化

准确识别 tracked / modified / staged / untracked / ignored 五种状态，支持 .gitignore 规则，忽略的目录不会展开。

📖

内置新手教程

附带完整的 Git 入门教程，从安装配置到分支管理、远程协作、撤销回滚，手把手带你成为 Git 高手。

🎯

零配置启动

双击 run.bat 即可运行，自动检测 Python 环境。未安装 Python？启动脚本会自动引导安装。

三区工作流

遵循 Git 经典的三区架构设计，让版本管理变得直观自然。

快速开始

30 秒上手

不需要安装任何额外依赖，双击就能用。

1

📥 下载项目

克隆或下载 Git 助手到本地任意文件夹。

2

🚀 双击运行

双击 run.bat，脚本会自动检测 Python 环境并启动。

3

📂 输入路径

在地址栏粘贴你的项目路径，按回车进入。

4

🎉 开始使用

双击暂存、填写说明、提交保存，像游戏存档一样管理代码！
