# Git 助手

可视化 Git 工具，无需记忆命令行，通过图形界面完成日常 Git 操作。

> 双击即用，无需安装任何依赖。

---

## 快速开始

```
双击 run.bat → 地址栏贴入项目路径 → 回车
```

若电脑未安装 Python，启动脚本会自动下载安装，全程无需手动操作。

---

## 环境要求

- **Git** — 操作 Git 仓库必需（[下载](https://git-scm.com/downloads)）
- **Python 3.8+** — 由启动脚本自动安装，无需手动准备
- **SSH 密钥**（可选）— 用于推送到 Gitee/GitHub

---

## 功能详解

### 📂 树形工作区

以目录树形式展示项目所有文件及其 Git 状态：

```
📝 工作区
├── 📁 src/
│   ├── main.c              ✅ 已跟踪
│   ├── utils/
│   │   └── helper.c        🟢 未跟踪
│   └── config.h            🟠 已修改
├── 📄 README.md             🔵 已暂存
└── 📁 build/                ⬜ 忽略
```

- ▶ **点箭头展开/收起文件夹**，查看内部文件状态
- **双击文件** = 暂存该文件
- 📁 忽略的目录（如 `node_modules`）不展开，不卡顿

### 📦 暂存区

| 操作 | 说明 |
|------|------|
| 📥 **暂存所有** | 一键 `git add .` |
| ↩️ **取消暂存** | 一键 `git restore --staged` |
| **双击暂存区文件** | 取消暂存该文件 |

### ✅ 已提交历史

- 查看 Git 提交记录（哈希值 + 提交说明）
- **选中版本 → 点击「恢复到选中版本」** = 安全回滚（使用 `git restore --source`，不修改历史）

### 🆕 一键新建 Git 仓库

在非 Git 仓库目录下，顶部自动显示提示条：
```
⚠️ 当前目录不是 Git 仓库    [🆕 新建 Git 仓库]
```
点击即可 `git init`，无需命令行。

### 📁 地址栏

- 输入路径 + 回车 = 切换到该目录
- 点击「浏览」= 文件夹选择器
- 自动识别 Git 仓库状态

### 🌐 Gitee 仓库管理

点底部 **「🌐 Gitee」** 按钮，打开远程仓库管理对话框：

```text
┌─ Gitee 仓库管理 ──────────────────────┐
│                                        │
│  ── 连接状态 ──                       │
│  ✅ 已连接到 Gitee                     │
│  分支: master                          │
│  远程: git@gitee.com:用户/仓库名.git   │
│  ✅ SSH 已连接  Hello 用户名!          │
│                                        │
│  [⬇️ 拉取]  [⬆️ 推送]  [🔄 刷新]     │
│                                        │
└────────────────────────────────────────┘
```

| 按钮 | 功能 | 对应命令 |
|------|------|---------|
| ⬇️ **拉取 (Pull)** | 从远程拉取最新代码 | `git pull` |
| ⬆️ **推送 (Push)** | 将本地提交推送到远程 | `git push` |
| 🔄 **刷新** | 重新检测 SSH 连接状态 | |

首次使用：
- 如果远程仓库未配置，自动弹出输入框填写 SSH 地址（支持 HTTPS 粘贴，自动转 SSH）
- 配置一次后，后续直接使用
