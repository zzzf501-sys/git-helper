# Git 助手

可视化 Git 工具，支持工作区/暂存区/已提交三区操作。
无需安装依赖，双击即可运行。

## 使用

```
双击 run.bat → 地址栏贴入项目路径 → 回车
```

若未安装 Python，启动脚本会自动提示并一键安装。

## 环境要求

- **Git**：操作 Git 仓库必需

> Python 由启动脚本自动安装，无需手动准备。

## 目录结构

```
git-helper/
├── run.bat              ← 启动入口
├── 启动Git助手.bat       ← 中文启动入口
├── readme.md            ← 本文件
├── .gitignore
└── src/
    ├── git-helper.py    ← 主程序
    └── git-tutorial.md  ← Git 教程
```

## 功能

- 📂 **树形工作区**：目录结构预览，文件 Git 状态一目了然
- 📦 **暂存区**：一键暂存 / 取消暂存
- ✅ **已提交历史**：查看提交记录，选中版本一键回滚
- 🆕 **一键新建 Git 仓库**
- 📁 **地址栏切换目录**，自动识别 Git 状态
