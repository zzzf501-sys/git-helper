# Git 助手

可视化 Git 工具，支持工作区/暂存区/已提交三区操作。
无需安装任何依赖，双击即用。

## 使用

双击 `启动Git助手.bat`（或 `run.bat`）→ 地址栏贴入项目路径 → 回车

## 一键安装环境

如果电脑没有 Python，启动脚本会自动提示安装：

1. 双击 `run.bat`
2. 按 `1` → 自动下载并安装 Python（自动加入 PATH）
3. 安装完成后自动启动 Git 助手

> 全部自动完成，无需手动操作。如果不方便自动安装，选 `2` 手动下载。

## 环境要求

| 软件 | 说明 |
|------|------|
| **Git** | 操作 Git 仓库必需（https://git-scm.com/downloads） |

## 目录结构

```
git-helper/
├── 启动Git助手.bat    ← 启动入口（双击运行）
├── readme.md          ← 本文件
└── src/
    ├── git-helper.py   ← 主程序
    └── git-tutorial.md ← Git 教程
```

## 功能

- 📂 **树形工作区**：目录结构预览，文件 Git 状态一目了然
- 📦 **暂存区**：一键暂存/取消暂存
- ✅ **已提交历史**：查看提交记录，选中版本一键回滚
- 🆕 **一键新建 Git 仓库**
- 📁 **地址栏切换目录**，自动识别 Git 状态

## 分享给他人

直接把整个 `git-helper/` 文件夹发过去就行。
对方只需安装 Python 和 Git 即可使用，无需任何配置。
